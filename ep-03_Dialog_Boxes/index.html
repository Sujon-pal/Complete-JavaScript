JavaScript Dialog Boxes ‚Äî alert, confirm, and prompt

Dialog boxes are built-in browser methods used to interact with users. They are simple, modal (block further code execution), and each serves a specific purpose.

1Ô∏è‚É£ alert() ‚Äî Show a simple message

Displays a message and waits for the user to click "OK". Does not return anything.

function showAlert() {
  alert("Welcome! This is an alert box.");
}

2Ô∏è‚É£ confirm() ‚Äî Ask for confirmation

Shows a message with "OK" and "Cancel". Returns true if the user clicks OK, false if Cancel.

function showConfirm() {
  let result = confirm("Do you want to continue?");
  console.log("User clicked:", result);
}

3Ô∏è‚É£ prompt() ‚Äî Ask for user input

Shows an input field with "OK" and "Cancel". Returns the text the user types or null if Cancel.

function showPrompt() {
  let name = prompt("What is your name?");
  console.log("User entered:", name);
}

‚ö° Combining all three together

This example uses all three dialog boxes in a flow: welcome alert ‚Üí confirm ‚Üí prompt ‚Üí final message.

function fullDemo() {
  alert("Welcome to our full demo!");

  let wantsToContinue = confirm("Do you want to continue?");

  if (wantsToContinue) {
    let userName = prompt("Great! What is your name?");

    if (userName !== null && userName.trim() !== "") {
      alert("Hello, " + userName + "! Nice to meet you.");
    } else {
      alert("You didn't enter your name, but that's okay!");
    }

  } else {
    alert("Maybe next time. Goodbye!");
  }
}

üß† Key points to remember

  • Modal: They pause other code until user responds.
  • Very limited styling: You cannot change colors, fonts, etc.
  • Simple use cases: Good for quick messages or small inputs.
  • For complex designs, use custom dialogs built with HTML, CSS, and JS.

‚úÖ Summary

  • alert(message) ‚Üí Show a simple message.
