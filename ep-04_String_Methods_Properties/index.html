📝 Template Literals, String Methods & Properties

📌 Strings are Indexed

Strings act like arrays of characters, so you can access each character by its index:

Expression: 'Hello World'[1]

Result: 'e'

Important: Strings are immutable → you can't change a character directly:

let str = 'Hello';
str[0] = 'Y'; // ❌ won't change 'H' to 'Y'

🎉 Template Literals

Template literals use backticks ` instead of quotes, and allow:

  • Multi-line strings without \n
  • Embedding variables: ${variable}
  • Embedding expressions: ${1+2}

Example:

let name = "Sujon";
let age = 20;
let message = `Hello, ${name}! Next year you'll be ${age+1}.`;
console.log(message); // Hello, Sujon! Next year you'll be 21.

📏 String Property

  • length — gives the number of characters

    • Example: "Hello".length // 5

🔧 Methods (Without Argument)

  • toUpperCase() → "hello".toUpperCase() → "HELLO"
  • toLowerCase() → "HELLO".toLowerCase() → "hello"
  • trim() → removes spaces from both ends
  • trimStart() → removes spaces from start only
  • trimEnd() → removes spaces from end only
  • repeat(n) → repeats the string n times

🛠 Methods (With Argument)

  • includes(substring) → checks if contains text
  • indexOf(substring) → returns position (or -1 if not found)
  • replace(old, new) → replaces first match
  • replaceAll(old, new) → replaces all matches
  • concat(otherString) → joins strings
  • padStart(length, padStr) → adds padding at start
  • padEnd(length, padStr) → adds padding at end
  • charAt(index) → returns character at index
  • charCodeAt(index) → returns ASCII code of character
  • split(separator) → splits into array
  • slice(start, end) → cuts part of string
  • startsWith(substring) → checks start
  • endsWith(substring) → checks end

✨ Extra Useful Methods

  • substring(start, end) → similar to slice but can’t use negative indexes
  • substr(start, length) → get part of string (deprecated but still works)
  • localeCompare(other) → compares two strings (returns -1, 0, or 1)
  • match(regex) → find matches using regular expressions
  • search(regex) → search by regex, returns index

🚀 Tips to Learn Easily

  • Read method names: they sound like what they do (trim, slice, replace)
  • Try them in browser console or online playground
  • Use template literals whenever you add variables or need multi-line text
