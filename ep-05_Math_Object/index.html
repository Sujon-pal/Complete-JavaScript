Introduction

Understand the Remainder Operator (%)

The remainder operator returns the remainder after division.

let remainder = 10 % 3; // 1

Understand the Exponentiation Operator (**)

The exponentiation operator raises the left-hand operand to the power of the right-hand operand.

let result = 2 ** 3; // 8

JavaScript Math Object

The Math object is a built-in object in JavaScript that provides properties and methods for mathematical constants and functions.

Math.sqrt() Method

Returns the square root of a number.

let result = Math.sqrt(16); // 4

Math.pow() Method

Returns the base raised to the exponent power.

let result = Math.pow(2, 3); // 8

Math.floor() Method

Rounds a number down to the nearest integer.

let result = Math.floor(4.9); // 4

Math.ceil() Method

Rounds a number up to the nearest integer.

let result = Math.ceil(4.1); // 5

Math.round() Method - Frequently Used!

Rounds a number to the nearest integer.

let result = Math.round(4.5); // 5

Math.random() Method - Most Used!

Returns a random number between 0 (inclusive) and 1 (exclusive).

let randomNumber = Math.random(); // e.g., 0.875

Use Math.random() and Math.floor() Together

Generate a random integer between two values.

let randomInt = Math.floor(Math.random() * 10); // 0 to 9

Understand Infinity and NaN in JavaScript

Infinity represents a number greater than any finite number.
NaN stands for "Not-a-Number", used when a value is not a valid number.

Mathematical Constants in JavaScript

The Math object provides constants such as Math.PI, Math.E, and others.

let piValue = Math.PI; // 3.141592653589793

Advanced Trigonometric Methods

JavaScript Math object provides methods for trigonometric calculations such as sine, cosine, and tangent.

let angle = Math.PI / 4;
let sinValue = Math.sin(angle); // 0.7071067811865475
let cosValue = Math.cos(angle); // 0.7071067811865476
let tanValue = Math.tan(angle); // 1

Logarithmic Functions

Logarithms are crucial for many algorithms, and JavaScript supports log calculations.

let logValue = Math.log(10); // Natural logarithm, approximately 2.302585
let log10Value = Math.log10(100); // Logarithm base 10, 2

Math.trunc() Method

The Math.trunc() method returns the integer part of a number by removing any fractional digits.

let truncatedValue = Math.trunc(4.9); // 4
