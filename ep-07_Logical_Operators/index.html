Logical Operators in JavaScript

1. AND Operator (&&) Example

The AND operator (&&) returns true if both operands are true. Otherwise, it returns false.

let age = 25;
let hasPermission = true;
let isMember = true;
console.log(age > 18 && hasPermission && isMember); // true
console.log(age > 30 && hasPermission && isMember); // false
console.log(age > 18 && !hasPermission && isMember); // false

2. OR Operator (||) Example

The OR operator (||) returns true if at least one operand is true. Otherwise, it returns false.

let age = 25;
let hasPermission = false;
let isMember = true;
console.log(age > 18 || hasPermission || isMember); // true
console.log(age < 18 || !hasPermission || isMember); // true
console.log(age < 18 || !hasPermission || !isMember); // false

3. NOT Operator (!) Example

The NOT operator (!) inverts the boolean value of an expression.

let hasPermission = false;
let isAdmin = true;
console.log(!hasPermission); // true
console.log(!isAdmin); // false
console.log(!(age > 18 && hasPermission)); // true

4. Combining Logical Operators Example

You can combine logical operators to create more complex conditions. Here's how:

let age = 25;
let isAdmin = false;
let hasPermission = true;
let isMember = true;
console.log((age > 18 && hasPermission) || (isAdmin && isMember)); // true
console.log((age < 30 || isAdmin) && !hasPermission); // false
console.log((age > 18 || !isAdmin) && (hasPermission || isMember)); // true

5. Truthy and Falsy Values with Logical Operators Example

In JavaScript, truthy values are non-zero numbers, non-empty strings, and objects. Falsy values include 0, null, undefined, and NaN.

let user = { name: "John" };
let age = 0;
let isLoggedIn = false;
console.log(user.name && age > 18 && isLoggedIn); // false
console.log(user.name && (age > 18 || isLoggedIn)); // "John"
console.log(!(age || isLoggedIn)); // true
