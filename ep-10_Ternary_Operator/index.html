🌟 JavaScript Ternary Operator - Beginner to Advanced

🔰 What is a Ternary Operator?

The ternary operator is a concise way to write conditional expressions. It's a shorthand for the if-else statement.

Syntax:

condition ? value_if_true : value_if_false;

✅ Simple Example


let isRaining = true;
let result = isRaining ? "Take an umbrella." : "Enjoy the sunshine!";
console.log(result); // Output: Take an umbrella.

🧠 Using Ternary for Number Check

Is the number even or odd?


let num = 7;
let result = (num % 2 === 0) ? "Even" : "Odd";
console.log("The number is " + result);

⚙️ Nested Ternary (Advanced)

Assign grades based on score:


let score = 78;
let grade = score >= 90 ? "A" :
            score >= 80 ? "B" :
            score >= 70 ? "C" :
            score >= 60 ? "D" : "F";
console.log("Grade:", grade); // Output: C

🎯 Ternary vs If-Else

Use ternary when:

  • ✔ Condition is simple
  • ✔ You need to assign a value based on a condition

Avoid ternary when:

  • ❌ Conditions are complex or deeply nested
  • ❌ Multiple steps or logic blocks needed

💡 Real-World Use: Display Greeting


let hour = new Date().getHours();
let greeting = hour < 12 ? "Good Morning" : hour < 18 ? "Good Afternoon" : "Good Evening";
console.log(greeting);

✅ Best Practices

  • ✅ Prefer readability over compactness
  • ✅ Use for return values, assignments, and JSX (React)
  • ✅ Avoid using multiple ternaries in one line

📌 Summary

  • condition ? true_value : false_value
  • Useful for quick checks and assignments
  • Can be nested but may reduce clarity
