JavaScript Objects Explained

Everything you need to know about JS objects !

1. What is an Object?

An object is a collection of properties. Each property is a key-value pair. 

const person = {
  name: "Sujon",
  age: 25,
  isStudent: true
};
ðŸ”¹ Keys are always strings. Values can be any data type: string, number, array, function, etc.

2. Creating Objects

  • Object Literal (most common)
  • Constructor Function
  • Object.create()
  • Class Syntax (ES6+)
// Object Literal
const car = {
  brand: "Toyota",
  model: "Corolla"
};

// Constructor
function Book(title, author) {
  this.title = title;
  this.author = author;
}
const b1 = new Book("JS Guide", "John");

// Object.create()
const base = { type: "base" };
const derived = Object.create(base);

// Class Syntax
class User {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
}
const u1 = new User("Sujon", 25);

3. Accessing Properties

console.log(person.name);       // Dot notation
console.log(person["age"]);     // Bracket notation
ðŸ’¡ Use bracket notation when keys are dynamic or have special characters.

4. Adding / Updating / Deleting Properties

person.email = "sujon@mail.com";    // Add
person.age = 26;                    // Update
delete person.isStudent;           // Delete

5. Nested Objects

const student = {
  name: "Sujon",
  address: {
    city: "Sylhet",
    zip: 3100
  }
};
console.log(student.address.city);

6. Looping through Objects

for (let key in person) {
  console.log(key + ": " + person[key]);
}

Object.keys(person);     // Array of keys
Object.values(person);   // Array of values
Object.entries(person);  // Array of [key, value]

7. Object Methods

const user = {
  name: "Sujon",
  greet: function() {
    return "Hello " + this.name;
  }
};
console.log(user.greet());

8. The "this" Keyword

"this" refers to the current object context.

const obj = {
  name: "Pal",
  show: function() {
    console.log(this.name);  // Refers to obj
  }
};

9. Object Destructuring

const { name, age } = person;
console.log(name);

10. Spread and Rest with Objects

const obj1 = { a: 1, b: 2 };
const obj2 = { ...obj1, c: 3 };   // Spread
function show({ a, ...rest }) {
  console.log(rest);
}

11. Object.freeze() and Object.seal()

const config = { debug: true };
Object.freeze(config);  // Can't modify or add
Object.seal(config);    // Can't add or delete, but can update

12. Object.assign()

Used to copy properties from one or more objects to another.

const target = {};
const source = { x: 10 };
Object.assign(target, source);

13. Comparing Objects

const objA = { x: 1 };
const objB = { x: 1 };
console.log(objA === objB); // false (different references)

14. Optional Chaining & Nullish Coalescing

const data = { user: { name: "Sujon" } };
console.log(data.user?.name);         // Safe access
console.log(data.age ?? "Unknown");   // Fallback value

15. JSON and Objects

const obj = { name: "Sujon" };
const str = JSON.stringify(obj);  // Convert to string
const back = JSON.parse(str);     // Convert to object
